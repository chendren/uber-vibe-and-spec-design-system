Are the slides just HTML?

Mostly yes — but with a few dependencies worth knowing about. The deck is a single HTML file (workshop/Claude Code Vibe and Spec Workshop.html) where every slide is plain static HTML markup (<section> elements). But it isn't 100% self-contained — it relies on:

  • deck-stage.js — the local script (in the workshop/ folder) that does the slide navigation, scaling, thumbnail rail, and print/PDF layout.
  • tweaks-panel.jsx + React/Babel — loaded from a CDN, powers the Tweaks panel (accent/section color, motif toggle).
  • Lucide — loaded from a CDN for the icons.
  • ../styles.css and ../assets/*.svg — the brand tokens and logos/motifs, which live one level up in the design-system root. This is why it matters that the whole project folder went to GitHub, not just the deck file.

So the content is just HTML (easy to edit text directly), but it needs those companion files and an internet connection (for the CDN scripts) to run as a presentation.
