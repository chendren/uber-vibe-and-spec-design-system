# Slides — teaching-deck templates

1280×720 (16:9) slide templates for live workshop sessions. Each is a standalone, directly-editable HTML file using the brand tokens and assets — drop your content in and present, or copy them into a deck.

| File | Use |
|---|---|
| `TitleSlide.html` | Opening card — logo, big serif title, presenter line. |
| `SectionSlide.html` | Module divider — big number + serif title on clay. |
| `ContentSlide.html` | Teaching point — heading, key points, a coach's-note callout. |
| `CodeDemoSlide.html` | Live demo — teaching point beside a Claude Code terminal. |
| `ComparisonSlide.html` | Vibe vs Spec, side by side. |
| `QuoteSlide.html` | Big testimonial on ink with the vibe→spec motif. |
| `ClosingSlide.html` | Sign-off — CTA + contact. |

## System
- Light slides on `--sand-50`; section/closing on brand color; quote on `--ink-900`.
- Serif (Newsreader) for titles, mono (JetBrains Mono) for eyebrows/labels/footers, sans for body.
- Keep the logo mark in a corner and a mono footer label for rhythm.
- Icons are Lucide (CDN). Text never below 24px at presentation scale.
